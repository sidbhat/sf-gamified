# Product Roadmap
## SF Joule Mario Quest Chrome Extension

**Last Updated**: 2025-12-07

---

## Vision

Transform SAP SuccessFactors training from boring documentation into engaging, game-like experiences that make users WANT to learn Joule capabilities.

---

## Phase 1: MVP (Week 1) ✅ Current Focus

**Goal**: Prove the concept with ONE working quest

### Deliverables
- ✅ AI-first development framework (.clinerules)
- ✅ Documentation structure (PRD, FEATURES, ARCHITECTURE)
- [ ] Working Chrome extension
- [ ] "View Cost Center" quest (3 steps)
- [ ] Demo + Real modes
- [ ] Mario-themed progress overlay
- [ ] Confetti celebration

### Success Criteria
- Extension loads on SF pages
- Quest completes successfully in demo mode
- Quest completes successfully in real mode
- User sees clear progress feedback

---

## Phase 2: Scale (Week 2-3)

**Goal**: Expand to 5 quests covering all Joule use case types

### Planned Quests

#### Informational (2 quests)
1. "View Cost Center" (Already in Phase 1)
2. "Top 5 Suppliers by Volume"

#### Navigational (1 quest)
3. "Navigate to Recruiting Dashboard"

#### Transactional (1 quest)
4. "Create SMART Goal"

#### Analytical (1 quest)
5. "Analyze Team Performance"

### Additional Features
- Multi-user support (10 user configs)
- Role-based quest filtering
- Progress persistence (resume interrupted quests)
- Enhanced shadow DOM support

---

## Phase 3: Advanced Features (Month 2)

**Goal**: Production-ready with enterprise features

### Features
- Analytics dashboard (quest completion stats)
- Quest history and replay
- Achievement system (badges)
- Team features:
  - Leaderboards
  - Shared quest libraries
  - Team challenges

### Technical Improvements
- AI-powered selector detection (handle UI changes)
- Performance optimization
- Error recovery mechanisms
- Comprehensive testing suite

---

## Phase 4: Community & Ecosystem (Month 3+)

**Goal**: Open-source and community-driven

### Features
- Visual quest builder (no-code)
- Quest marketplace (community submissions)
- Custom branding support
- Multi-language support
- Integration with SAP Learning Management System

### Distribution
- Chrome Web Store publication
- GitHub open-source release
- Documentation website
- Video tutorials

---
