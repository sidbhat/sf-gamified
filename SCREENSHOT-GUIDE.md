## ğŸ–¥ï¸ How to Capture Screenshots

### Method 1: Chrome DevTools (Recommended)

**Step 1: Set Up Extension**
```bash
1. Load extension in Chrome (chrome://extensions)
2. Navigate to SAP SuccessFactors page
3. Click extension icon to open quest selection
```

**Step 2: Configure DevTools**
```bash
1. Open DevTools (F12 or Cmd+Option+I)
2. Click "Toggle device toolbar" (Cmd+Shift+M)
3. Select "Responsive" at top
4. Set dimensions: 1280 x 800
5. Zoom: 100%
```

**Step 3: Capture**
```bash
1. Position the view perfectly
2. Open Command Palette (Cmd+Shift+P / Ctrl+Shift+P)
3. Type "screenshot"
4. Select "Capture screenshot"
5. Save as PNG
```

**Step 4: Repeat**
- Screenshot 1: Quest selection (both tabs)
- Screenshot 2: Click a quest, capture in-progress
- Screenshot 3: Complete quest, capture success screen
- Screenshot 4: Quest map with mixed states
- Screenshot 5: Hover over quest node (if possible)
