# Install Joule Quest Locally (Developer Mode)

## What's in the ZIP?

✅ **Complete Extension with Source Code**
- All JavaScript source files (src/)
- All configuration files (manifest.json, selectors, quests)
- All i18n translations (11 languages)
- All assets (icons, screenshots, logo)
- **NO** git files, docs, scripts, or temporary files

**File Size:** ~2.0 MB
**Package:** joule-quest-v1.1.1.zip

---

## Installation Steps

### 1. Download the Package
- Download **joule-quest-v1.1.1.zip** to your computer

### 2. Extract the Files
- Unzip the package to a folder (e.g., `joule-quest-v1.1.1/`)
- Keep this folder - don't delete it (Chrome loads from this location)

### 3. Open Chrome Extensions
- Open Chrome browser
- Navigate to: **chrome://extensions/**
- Or: Menu → Extensions → Manage Extensions

### 4. Enable Developer Mode
- Toggle **"Developer mode"** switch in top-right corner
- This allows loading unpacked extensions

### 5. Load the Extension
- Click **"Load unpacked"** button
- Browse to the extracted folder (joule-quest-v1.1.1/)
- Select the folder and click "Select"

### 6. Verify Installation
- ✅ Joule Quest icon appears in Chrome toolbar
- ✅ Extension listed in chrome://extensions/
- ✅ Status shows "Enabled"

## Differences: Local Install vs Chrome Web Store

### Local Install (Developer Mode):
- ✅ Install immediately (no review wait)
- ✅ Full source code visible
- ✅ Can modify and test changes
- ⚠️ Shows "Developer mode" warning banner
- ⚠️ Must manually update for new versions
- ⚠️ No auto-updates

### Chrome Web Store:
- ⏳ Must wait for Google review (~1-3 days)
- ✅ Source code still visible (Chrome requirement)
- ✅ Auto-updates for all users
- ✅ No warning banners
- ✅ Official distribution
- ✅ Better for end users
